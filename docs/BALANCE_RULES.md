# BALANCE_RULES.md — 밸런스 규칙 및 튜닝 포인트

## 1. 목표 분포(초기)
- 팀 득점: 평균 1.2 ~ 1.6
- 경기 총 득점: 평균 2.2 ~ 2.8
- PC 득점 관여(초반): 0.2 ~ 0.6 / 경기
- 평균 평점: 5.8 ~ 7.2
- 부상(경미 이상): 10경기당 1~2회 수준(무리하면 증가)

## 2. 튜닝 포인트 목록
- 이벤트별 Base 성공 확률
- 스탯 가중치(이벤트별 weight)
- 피로 페널티 곡선
- 자신감 보너스
- 상대 난이도(수비/공격 레이팅 → contextModifier)

## 3. 상대 난이도 반영(예시)
- OppDefenseRating 40~90을 0..1로 정규화
- 득점형 이벤트에 -0.00 ~ -0.12 정도의 modifier 적용
- 압박 이벤트 성공률에 +0.00 ~ +0.08 적용(상대가 강하면 압박 성공 어려움)

## 4. 훈련 밸런스(예시)
- 훈련 1회로 스탯 +0~+2(확률)
- 강훈련은 피로 +12~+18
- 휴식은 피로 -15~-25, 자신감 소폭 회복
- 재활은 부상 회복 가중

## 5. 신뢰 임계치(예시)
- 0~35: 주로 벤치/후반 교체, 하이라이트 8~10개
- 36~70: 로테이션/선발 혼합, 하이라이트 10~12개
- 71~100: 주전, 하이라이트 12~14개

## 6. 회귀 방지 체크리스트
- 1000경기 시뮬레이션 결과를 스냅샷으로 저장
- 주요 KPI가 ±10% 이상 바뀌면 경고
