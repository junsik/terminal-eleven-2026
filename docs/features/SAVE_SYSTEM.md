# 저장/불러오기 시스템 💾

Hive 기반의 로컬 저장 시스템입니다.

---

## 저장 방식

### 자동 저장
다음 상황에서 자동 저장됩니다:
- 새 게임 시작
- 경기 종료 후
- 훈련 완료 후
- 라운드 진행 시

### 수동 저장
현재 버전에서는 자동 저장만 지원합니다.

---

## 저장 데이터

### GameSnapshot 구조
```dart
{
  version: 1,
  savedAt: "2024-01-20T12:00:00Z",
  gameState: "home",
  pc: { ... },           // 플레이어 정보
  season: { ... },       // 시즌 정보
  activeMatch: null,     // 진행 중 경기
  weeklyActionsRemaining: 3
}
```

---

## 백업 시스템

- **이중 저장**: 메인 + 백업
- 메인 실패 시 백업에서 복구
- JSON 형식으로 직렬화

---

## 이어하기

### 로비 화면
저장된 게임이 있으면 "이어하기" 버튼 표시

```
┌─────────────────────────┐
│   🎮 MUD 축구           │
│                         │
│   [이어하기]  ← 저장됨  │
│   [새 게임]             │
└─────────────────────────┘
```

### 저장 삭제
새 게임 시작 시 기존 저장 삭제 확인
