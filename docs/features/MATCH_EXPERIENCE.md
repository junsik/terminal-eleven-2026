경기당 **10-14개**의 하이라이트가 생성됩니다.

## 확률 시스템

### 성공 확률 공식
```
p = Base + (스탯보너스 × 0.3) 
    - 피로페널티 
    + 자신감보너스 
    + 모멘텀보너스
    - 상대난이도
```

### 피로 페널티
| 피로 | 페널티 |
|------|--------|
| 0-60 | 0% |
| 61-80 | (피로-60) × 0.3% |
| 81-100 | 6% + (피로-80) × 0.6% |

### 모멘텀 시스템
- **연속 3회 성공** 시 "HOT" 상태 → +5% 보너스
- 모멘텀 범위: -3 ~ +3 (각 ±2% 영향)

100+ 한국어 템플릿으로 다양한 상황 연출

```
🔥 극적인 골!!!! 드라마다!!!
```
