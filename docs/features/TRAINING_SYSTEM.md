# 훈련 및 성장 시스템 🏋️

선수의 능력치를 향상시키는 훈련 시스템입니다.

---

## 훈련 종류

### 기술 훈련
| 훈련 | 향상 스탯 | 피로 증가 |
|------|----------|----------|
| 슈팅 훈련 | 슈팅 +1~2 | +15 |
| 패스 훈련 | 패스 +1~2 | +12 |
| 드리블 훈련 | 볼컨트롤 +1~2 | +12 |
| 위치 선정 | 포지셔닝 +1~2 | +10 |

### 기초 체력
| 훈련 | 향상 스탯 | 피로 증가 |
|------|----------|----------|
| 체력 훈련 | 스태미나 +1~2 | +18 |
| 침착성 훈련 | 침착성 +1~2 | +8 |

### 회복
| 훈련 | 효과 | 피로 감소 |
|------|------|----------|
| 휴식 | 피로 회복 | -20 |
| 재활 | 부상 회복 촉진 | -10 |

---

## 주간 제한

- **주당 3회** 훈련/휴식 가능
- 경기 후 다음 라운드까지 리셋

---

## 레벨업 시스템

### 경험치 획득
| 액션 | XP |
|------|-----|
| 경기 출전 | +10 |
| 골 | +20 |
| 어시스트 | +15 |
| 평점 7+ | +10 |
| 훈련 완료 | +5 |

### 레벨별 필요 XP
| 레벨 | 필요 XP |
|------|---------|
| 1→2 | 100 |
| 2→3 | 200 |
| 3→4 | 350 |
| ... | +150씩 증가 |

---

## 선수 스탯

7가지 핵심 스탯 (0-100)

| 스탯 | 설명 | 주요 상황 |
|------|------|----------|
| 속도 (Pace) | 스피드 | 역습, 침투 |
| 슈팅 | 마무리 | 1:1, 중거리 |
| 패스 | 배급력 | 역습, 어시스트 |
| 볼컨트롤 | 기술 | 턴, 경합 |
| 위치선정 | 공간활용 | 침투, 수비 |
| 스태미나 | 체력 | 압박, 후반전 |
| 침착성 | 멘탈 | 클러치, 압박 |
