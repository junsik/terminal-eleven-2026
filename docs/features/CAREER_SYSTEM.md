# 커리어 및 신뢰도 시스템 📈

장기적인 커리어 성장과 감독 신뢰 관리입니다.

---

## 신뢰도 (Trust)

감독에게 얼마나 신뢰받는지를 나타냅니다.

### 신뢰도 영향
| 범위 | 효과 |
|------|------|
| 0-35 | 벤치/후반 교체, 하이라이트 8-10개 |
| 36-70 | 로테이션/선발 혼합, 10-12개 |
| 71-100 | 주전, 12-14개 |

### 신뢰도 변화
```
trustΔ = (평점 - 6.0) × 4 + 행동 가산
```

| 액션 | 변화 |
|------|------|
| 평점 8.0 | +8 |
| 평점 5.0 | -4 |
| 코치 지시 수용 | +3 |
| 코치 지시 무시 | -5 |
| 무리한 플레이 | -2 |

## K리그 시즌

### 리그 구성
- **10개 팀** 참가
- **더블 라운드 로빈** (18라운드)
- **홈/원정** 각 9경기

### 순위 결정
1. 승점 (승 3점, 무 1점)
2. 골득실
3. 다득점
