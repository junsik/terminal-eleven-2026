# 선수 성장

## XP 시스템

### XP 획득
경기 후 평점에 따라 XP를 획득합니다:

```
XP 획득량 = 평점 × 10
```

| 평점 | XP |
|------|-----|
| 9.0 | 90 |
| 7.5 | 75 |
| 6.0 | 60 |
| 5.0 | 50 |

### 레벨업
- XP가 일정량 누적되면 레벨업
- 레벨업 시 스탯 포인트 획득
- 최대 레벨: 제한 없음

## 신뢰도 시스템

### 신뢰도란?
감독의 신뢰를 나타내는 수치입니다. 높을수록 주전 기회가 많아집니다.

### 신뢰도 변화
```
신뢰도 변화 = (평점 - 6.0) × 4
```

| 평점 | 신뢰도 변화 |
|------|------------|
| 8.0 | +8 |
| 7.0 | +4 |
| 6.0 | ±0 |
| 5.0 | -4 |
| 4.0 | -8 |

### 신뢰도 효과
- 높은 신뢰도: 더 많은 출전 기회, 좋은 포지션
- 낮은 신뢰도: 벤치 위기, 임대/방출 가능성

## 커리어 통계

### 누적 기록
- 출전 경기 수
- 득점
- 어시스트
- 평균 평점

### 시즌 기록
- 시즌별 성적 저장
- 리그 개인 순위 (득점왕, 도움왕 등)

## 부상과 회복

### 부상 발생
- 과도한 피로 + 격한 플레이 = 부상 위험 증가
- 부상 시 일정 주 동안 경기 결장

### 부상 중 행동
- **재활**: 부상 회복 가속
- **휴식**: 피로 회복 + 느린 부상 회복

### 결장 시
- 경기에 출전 불가
- 팀 경기는 자동으로 시뮬레이션
- 평점/XP 획득 없음

## 성장 전략

### 초반 (Lv.1~10)
- 주요 스탯 2~3개 집중 육성
- 훈련 강도 "보통"으로 안정적 성장

### 중반 (Lv.11~20)
- 부족한 스탯 보완
- Composure 중요성 증가 (클러치 대비)

### 후반 (Lv.21+)
- 균형 잡힌 올라운더 또는 특화 스페셜리스트
- 시즌 우승 도전
