# 모멘텀 시스템

## 개요

모멘텀은 경기의 **"분위기"**를 나타내는 수치입니다. 좋은 플레이를 하면 모멘텀이 오르고, 실수하면 내려갑니다.

```
모멘텀 범위: -3 ~ +3
```

## 모멘텀 표시

경기 화면 상단의 **모멘텀 바**에서 현재 상태를 확인할 수 있습니다:

| 모멘텀 | 상태 | 효과 |
|--------|------|------|
| +2 ~ +3 | 🔥 DOMINATING | 성공률 +8~12% |
| +1 | GOOD FLOW | 성공률 +4% |
| 0 | NEUTRAL | 기본 |
| -1 | UNDER PRESSURE | 성공률 -4% |
| -2 ~ -3 | ⚠️ CRITICAL | 성공률 -8~12% |

## 모멘텀 변화

### 상승 요인
| 행동 | 모멘텀 변화 |
|------|------------|
| 골 성공 | +2 |
| 일반 성공 | +1 |
| 수비 성공 | +1 |
| 클러치 성공 | +3 |

### 하락 요인
| 행동 | 모멘텀 변화 |
|------|------------|
| 슈팅 실패 | -1 |
| 치명적 실수 | -2 |

## HOT & COLD 상태

### 🔥 HOT 상태
- **조건**: 3번 연속 성공
- **효과**: 추가 성공률 +8%
- **회복**: 실패 시 연속 성공 절반으로 감소 (완만한 하락)

### 🥶 COLD 상태
- **조건**: 2번 연속 실패
- **효과**: 성공률 -5%
- **회복**: 성공 시 연속 실패 절반으로 감소 (완만한 회복)

## 전략적 활용

### 모멘텀이 높을 때
- 위험한 플레이로 높은 보상 노리기
- 파워샷, 커브샷 등 고위험 선택 시도

### 모멘텀이 낮을 때
- 안전한 플레이로 연속 성공 쌓기
- **전술 외침**으로 모멘텀 회복
