# 클러치 타임

## 개요

**클러치 타임**은 경기 막판의 긴장감 넘치는 순간을 표현하는 시스템입니다.

## 발동 조건

클러치 타임은 다음 조건을 **모두** 만족할 때 활성화됩니다:

```
✓ 경기 시간 80분 이후
✓ 점수 차이 1점 이하 (동점 또는 1점차)
```

## 클러치 타임 효과

### Composure 스탯 가중치 증가
- 평소: Composure의 일반적인 영향
- 클러치 타임: **Composure 가중치 2배**

### 클러치 찬스 하이라이트
- 클러치 타임에만 발생하는 특별한 기회
- 성공 시 높은 평점 보상
- 실패 시 드라마틱한 연출

## 클러치 타임 전략

### 높은 Composure 선수
- 클러치 상황에서 우위
- 막판 뒤집기에 유리

### 낮은 Composure 선수
- 안전한 플레이 선택 권장
- 무리한 시도 자제

## 클러치 결과 메시지

### 성공 시
```
⭐ 결정적인 순간! 침착하게 해냈다!
⭐ 클러치!! 승부사의 본능이 빛났다!
⭐ 이 순간을 위해 기다렸다! 역전 골!
```

### 실패 시
```
💔 아... 아쉽다! 마지막 기회였는데!
💔 운명의 장난인가... 놓쳤다.
💔 너무 아쉽다... 이게 축구다.
```
