# 경기 시스템

## 경기 흐름

Terminal Eleven의 경기는 **하이라이트 기반**으로 진행됩니다. 90분 전체를 시뮬레이션하는 대신, 플레이어에게 중요한 순간만 추출하여 보여줍니다.

```
경기 시작 → 하이라이트 1 → 선택 → 결과 → 하이라이트 2 → ... → 경기 종료
```

## 하이라이트 종류

### 공격 상황
| 타입 | 설명 | 주요 스탯 |
|------|------|----------|
| **1 vs 1** | 골키퍼와 일대일 | Shooting, Composure |
| **박스 가장자리 슛** | 페널티 박스 외곽 슛 | Shooting, Positioning |
| **카운터 어택** | 역습 상황 | Pace, Passing |
| **스루패스 찬스** | 결정적 패스 기회 | Passing, Positioning |
| **세트피스 리바운드** | 세트피스 후 2차 기회 | Shooting |

### 수비 상황
| 타입 | 설명 | 주요 스탯 |
|------|------|----------|
| **프레싱** | 압박 수비 | Stamina, Positioning |
| **수비 커버** | 역습 대응 | Pace, Positioning |
| **루즈볼 경합** | 공 소유권 다툼 | Dribbling, Stamina |

### 특수 상황
| 타입 | 설명 |
|------|------|
| **페널티킥** | PK 상황 - 다양한 킥 옵션 |
| **클러치 찬스** | 경기 막판 결정적 기회 |

## 선택지 (Commands)

각 하이라이트에서 플레이어는 행동을 선택합니다:

### 기본 선택지
| 선택 | 효과 |
|------|------|
| **안전한 플레이** | 성공률 +15%, 보상 낮음 |
| **일반 플레이** | 기본 확률 |
| **위험한 플레이** | 성공률 -10%, 보상 높음 |

### 슈팅 옵션
| 선택 | 리스크 | 설명 |
|------|--------|------|
| **일반 슛** | 보통 | 기본 슈팅 |
| **파워샷** | 높음 | 강한 슛, 리스크 높음 |
| **커브샷** | 높음 | 휘어지는 슛 |
| **칩샷** | 매우 높음 | 골키퍼 넘기기 |
| **파넨카** | 극히 높음 | PK 전용, 최고 리스크 |
