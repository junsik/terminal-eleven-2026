# 세트피스 옵션

## 개요

페널티킥이나 프리킥 상황에서 다양한 **킥 옵션**을 선택할 수 있습니다. 각 옵션은 서로 다른 리스크와 리턴을 가집니다.

## 슈팅 옵션

| 옵션 | 리스크 | 성공률 보정 | 특징 |
|------|--------|------------|------|
| **일반 슛** | 보통 | 기본 | 안정적인 선택 |
| **파워샷** | 높음 | -10% | 강한 슛, 골키퍼 반응 어려움 |
| **커브샷** | 높음 | -10% | 휘어지는 궤적 |
| **칩샷** | 매우 높음 | -15% | 골키퍼 넘기기 |
| **너클샷** | 높음 | -12% | 불규칙한 궤적 |
| **파넨카** | 극히 높음 | -20% | 최고 리스크, 최고 멋 |

## 페널티킥

페널티킥 상황에서는 심리전이 중요합니다:

### 선택 전략
```
일반 슛 → 안전하게 득점
파워샷 → 막더라도 튕겨낼 가능성
파넨카 → 성공하면 영웅, 실패하면...
```

### 파넨카 킥
- **최고 리스크**: 성공률 -20%
- **성공 시**: 평점 대폭 상승 + 특별 멘트
- **실패 시**: 굴욕적인 실패 연출

## 세트피스 리바운드

세트피스 실패 후에도 **리바운드 찬스**가 발생할 수 있습니다:

- 골키퍼가 막았지만 공이 튀어나옴
- 빠른 반응으로 추가 슈팅 기회
- 별도의 하이라이트로 처리

## 전략 가이드

### 이길 때
- 안전한 옵션으로 리드 유지

### 질 때
- 고위험 옵션으로 역전 시도
- 클러치 타임이면 Composure 의존

### 동점일 때
- 상황에 따라 유연하게 선택
- 모멘텀이 좋으면 도전적으로
