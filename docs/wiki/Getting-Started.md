# 게임 시작하기

## 새 게임 시작

1. 앱을 실행하면 로비 화면이 나타납니다
2. "새 게임" 버튼을 눌러 새로운 커리어를 시작합니다
3. 선수 이름을 입력하고 시작!

## 기본 개념

### 주간 행동 (Weekly Actions)
- 매주 **3회**의 행동 기회가 주어집니다
- 훈련, 휴식 등을 선택할 수 있습니다
- 경기는 별도의 행동으로 자동 진행됩니다

### 시즌 구조
- 리그는 **8개 팀**으로 구성
- 총 **14라운드** (홈 & 어웨이)
- 순위에 따라 시즌 종료 보상이 달라집니다

## 선수 스탯

| 스탯 | 설명 |
|------|------|
| **Pace** | 스피드, 빠른 돌파 성공률 |
| **Shooting** | 슈팅 정확도, 골 결정력 |
| **Passing** | 패스 성공률, 어시스트 |
| **Dribbling** | 드리블 성공률, 볼 컨트롤 |
| **Positioning** | 위치 선정, 찬스 생성 빈도 |
| **Stamina** | 체력, 피로 회복 속도 |
| **Composure** | 침착성, 클러치 상황 성공률 |

## 선수 상태

| 상태 | 범위 | 설명 |
|------|------|------|
| **피로도** | 0~100 | 높을수록 경기력 하락, 부상 위험 증가 |
| **자신감** | -3~+3 | 성공 확률에 영향 |
| **부상** | 0~N주 | 부상 중에는 경기 결장 |

## 첫 주 추천 루틴

1. **훈련 2회**: 슈팅 또는 패스 훈련으로 주요 스탯 향상
2. **휴식 1회**: 피로도 관리로 경기 대비
3. **경기 진행**: 다음 라운드 경기에 출전
