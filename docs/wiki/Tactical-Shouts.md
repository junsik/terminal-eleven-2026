# 전술 외침

## 개요

**전술 외침**은 하이라이트 사이에 동료들에게 지시를 내려 모멘텀을 조절하는 시스템입니다.

## 외침 종류

| 외침 | 효과 | 설명 |
|------|------|------|
| **격려** 🗣️ | 모멘텀 +1 | "할 수 있어!" - 동료들을 격려 |
| **요구** 🗣️ | 모멘텀 +2 | "정신 차려!" - 강하게 지시 |
| **진정** 🗣️ | 모멘텀 -1 | "침착해!" - 흥분을 가라앉힘 |

## 사용 제한

### 쿨다운
- **하이라이트당 1회**만 사용 가능
- 같은 하이라이트에서 중복 사용 불가

### 전략적 사용
- 모멘텀이 낮을 때 **격려/요구**로 회복
- 모멘텀이 너무 높아 오버페이스 우려 시 **진정**

## UI 위치

경기 화면에서 하이라이트가 표시될 때 하단에 전술 외침 버튼이 나타납니다:

```
┌─────────────────────────┐
│  [하이라이트 내용]       │
│                         │
│  ┌────┐ ┌────┐ ┌────┐  │
│  │격려│ │요구│ │진정│  │
│  └────┘ └────┘ └────┘  │
└─────────────────────────┘
```

## 전략 가이드

### 연속 실패 시 (COLD 상태)
1. **요구** 외침으로 모멘텀 급상승
2. 안전한 플레이로 연속 성공 노리기
3. COLD 상태 탈출

### 경기 막판 역전 필요 시
1. **격려/요구**로 모멘텀 최대화
2. 클러치 타임 진입 전 준비
3. 고위험 선택지 도전

### 리드 중 안정적 마무리
1. **진정** 외침으로 페이스 조절
2. 안전한 플레이 선택
3. 무리한 도전 자제
