# TEST_PLAN.md — 테스트/검증 계획

## 1. 단위 테스트(Unit)
- 상태 머신 전이 테스트(GameState/MatchPhase)
- 확률 함수 clamp/페널티 곡선 테스트
- 평점 누적/정규화 테스트
- 스냅샷 직렬화/역직렬화 테스트(version 포함)

## 2. 시뮬레이션 테스트(Simulation)
- AutoPlayBot로 1,000경기 실행
- KPI 산출:
  - 총 득점 평균/분산
  - PC 득점 관여
  - 평균 평점 및 분포
  - 부상 발생률
- KPI가 목표 범위 밖이면 실패 처리(회귀 감지)

## 3. 시드 재현 테스트
- seed + 커맨드 시퀀스가 동일하면 결과가 동일해야 함
- 엔진 변경 시도 시, 의도된 변경인지 확인 가능

## 4. UI 스모크 테스트
